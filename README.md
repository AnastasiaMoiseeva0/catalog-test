# Проект "Менеджер Курсов"

## Описание

Этот проект реализует приложение "Менеджер Курсов", которое позволяет пользователям просматривать и фильтровать курсы по категориям и поисковым запросам. Пользователи могут искать конкретные курсы и динамически фильтровать результаты с помощью тегов, представляющих разные категории.

## Функции

- **Динамическая Фильтрация**: Фильтрация курсов по категории и поисковому запросу в реальном времени.
- **Система Тегов**: Отображение категорий с количеством доступных курсов в каждой категории.
- **Адаптивный Дизайн**: Удобный интерфейс, который адаптируется под разные размеры экранов.

## Используемые Технологии

- **JavaScript**: Основной язык программирования для логики приложения.
- **HTML/CSS**: Структура и стиль приложения.
- **ES6 Модули**: Использование современных функций JavaScript и модульного импорта.
